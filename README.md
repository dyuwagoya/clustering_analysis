# Clustering Analysis

## Introduction
The objective of this project is to enhance the efficiency of the supply chain by clustering similar products together.

## Project Steps Overview

1. Assessing data
2. Scaling and normalizing data
3. Dendogram
4. Cluster Analysis

## Algorithms

* Dendogram
* Hierarchical Clustering
